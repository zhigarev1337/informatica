Урок 6. Объекты алгоритмов. Алгоритмическая конструкция «следование»

Перечень рассматриваемых вопросов:

  • Понятие объектов алгоритмов.
  • Алгоритмическая конструкция «следование» и её особенности.
  • Примеры использования конструкции «следование» в алгоритмах.

Тезаурус:

Объекты алгоритмов – это данные, с которыми работают алгоритмы.

Следование – это последовательное выполнение действий одно за другим.

Теоретический материал для самостоятельного изучения

Алгоритмы работают с различными объектами, которые представляют собой данные. Объекты могут быть различного типа, например, числа, строки, массивы и другие структуры данных. В процессе выполнения алгоритма эти объекты могут изменяться в соответствии с заданной последовательностью действий.


Алгоритмическая конструкция «следование»

Конструкция «следование» представляет собой одну из основных структур управления в алгоритмах, при которой действия выполняются последовательно одно за другим. Это самая простая и интуитивно понятная конструкция, часто используемая для записи алгоритмов.

Пример использования конструкции «следование»:

  1. Прочитать значение переменной a.
  2. Прочитать значение переменной b.
  3. Вычислить сумму a + b и сохранить результат в переменную sum.
  4. Вывести значение переменной sum на экран.

Примеры алгоритмов с использованием конструкции «следование»

Пример 1: Алгоритм вычисления среднего арифметического двух чисел

Задача: Вычислить среднее арифметическое двух чисел.

Алгоритм:

  1. Прочитать значение переменной x.
  2. Прочитать значение переменной y.
  3. Вычислить сумму x + y.
  4. Разделить полученную сумму на 2 и сохранить результат в переменную average.
  5. Вывести значение переменной average на экран.

Пример 2: Алгоритм нахождения максимального из двух чисел

Задача: Найти максимальное из двух чисел.

Алгоритм:

  1. Прочитать значение переменной a.
  2. Прочитать значение переменной b.
  3. Если a > b, то сохранить значение a в переменную max, иначе сохранить значение b в переменную max.
  4. Вывести значение переменной max на экран.

Заключение

Понимание алгоритмической конструкции «следование» и умение работать с объектами алгоритмов является фундаментальной основой для написания эффективных и правильных алгоритмов. Конструкция «следование» позволяет четко и логично представлять последовательность действий, необходимых для решения различных задач.
