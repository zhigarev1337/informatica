Урок 11. Запись линейных алгоритмов на языке программирования Паскаль

Перечень рассматриваемых вопросов:

  • Понятие линейных алгоритмов.
  • Синтаксис записи линейных алгоритмов на языке Паскаль.
  • Примеры программ с линейными алгоритмами.

Тезаурус:

Линейный алгоритм – это алгоритм, в котором все команды выполняются последовательно, одна за другой.

Теоретический материал для самостоятельного изучения

Линейный алгоритм – это алгоритм, в котором все команды выполняются последовательно, одна за другой. Такие алгоритмы являются самыми простыми и понятными, так как их выполнение не зависит от условий или повторений.


Синтаксис записи линейных алгоритмов на языке Паскаль

Для записи линейных алгоритмов на языке программирования Паскаль используются следующие основные конструкции:

  • Объявление программы: Каждая программа начинается с ключевого слова program, за которым следует имя программы.
  • Объявление переменных: Переменные объявляются с использованием ключевого слова var.
  • Начало и конец программы: Основная часть программы заключается между ключевыми словами begin и end.
  • Операторы присваивания: Значения переменных присваиваются с использованием оператора :=.
  • Ввод и вывод данных: Для ввода данных используется процедура readln, для вывода данных – процедура writeln.

Примеры программ с линейными алгоритмами

Пример 1: Программа для вычисления суммы двух чисел

program Sum;
var
    a, b, sum: integer;
begin
    writeln('Введите два числа:');
    readln(a, b);
    sum := a + b;
    writeln('Сумма: ', sum);
end.

Пример 2: Программа для нахождения среднего арифметического двух чисел

program Average;
var
    a, b: integer;
    avg: real;
begin
    writeln('Введите два числа:');
    readln(a, b);
    avg := (a + b) / 2;
    writeln('Среднее арифметическое: ', avg:0:2);
end.

Пример 3: Программа для вычисления площади прямоугольника

program RectangleArea;
var
    length, width, area: integer;
begin
    writeln('Введите длину и ширину прямоугольника:');
    readln(length, width);
    area := length * width;
    writeln('Площадь прямоугольника: ', area);
end.

Заключение

Линейные алгоритмы являются основой для изучения программирования. Понимание их структуры и синтаксиса записи на языке Паскаль позволяет создавать простые и понятные программы, которые выполняются последовательно и без условий или циклов.
