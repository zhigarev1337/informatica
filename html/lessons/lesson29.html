Урок 5. Одномерные массивы целых чисел

Перечень рассматриваемых вопросов:

  • Понятие одномерного массива.
  • Инициализация и заполнение массива.
  • Основные операции над массивами.
  • Примеры задач с использованием массивов.

Тезаурус:

Одномерный массив – это структура данных, представляющая собой упорядоченный набор элементов одного типа, доступ к которым осуществляется по индексу.

Теоретический материал для самостоятельного изучения

Одномерный массив – это структура данных, которая позволяет хранить и обрабатывать набор однотипных элементов. Каждый элемент массива имеет свой индекс, с помощью которого осуществляется доступ к элементу. Индексация в массивах обычно начинается с нуля.


Инициализация и заполнение массива

Инициализация массива – это процесс выделения памяти для массива и задания начальных значений его элементов. Заполнение массива может осуществляться различными способами:

  • Явная инициализация при объявлении:
    • int arr[5] = {1, 2, 3, 4, 5};
  • Инициализация с помощью цикла:
    • int arr[5];
for (int i = 0; i < 5; i++) {
    arr[i] = i + 1;
}

Основные операции над массивами

С массивами можно выполнять различные операции, такие как:

  • Обращение к элементам массива: доступ к элементу массива осуществляется по его индексу. Например, arr[2] возвращает третий элемент массива arr.
  • Изменение значений элементов массива: присваивание нового значения элементу массива по индексу. Например, arr[2] = 10; изменяет третий элемент массива arr на 10.
  • Поиск элементов в массиве: можно искать элементы по значению или по индексу. Например, поиск минимального или максимального значения в массиве.
  • Сортировка массива: упорядочение элементов массива по возрастанию или убыванию. Например, сортировка методом пузырька.
  • Вывод элементов массива: можно выводить элементы массива на экран, используя цикл.

Примеры задач с использованием массивов

Массивы широко используются для решения различных задач. Примеры задач:

  • Нахождение суммы элементов массива:
    • int sum = 0;
for (int i = 0; i < 5; i++) {
    sum += arr[i];
}
  • Поиск минимального элемента в массиве:
    • int min = arr[0];
for (int i = 1; i < 5; i++) {
    if (arr[i] < min) {
        min = arr[i];
    }
}
  • Сортировка массива методом пузырька:
    • for (int i = 0; i < 5-1; i++) {
    for (int j = 0; j < 5-i-1; j++) {
        if (arr[j] > arr[j+1]) {
            int temp = arr[j];
            arr[j] = arr[j+1];
            arr[j+1] = temp;
        }
    }
}

Заключение

Одномерные массивы являются важной структурой данных, которая позволяет эффективно хранить и обрабатывать наборы однотипных элементов. Понимание работы с массивами и умение выполнять основные операции над ними является фундаментальным навыком программирования.
