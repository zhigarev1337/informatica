Урок 12. Запись ветвлений на языке Паскаль

Перечень рассматриваемых вопросов:

  • Понятие ветвлений в алгоритмах.
  • Синтаксис записи ветвлений на языке Паскаль.
  • Примеры программ с ветвлениями.

Тезаурус:

Ветвление – это конструкция, которая позволяет выполнять разные действия в зависимости от выполнения условий.

Теоретический материал для самостоятельного изучения

Ветвление – это конструкция, которая позволяет выполнять разные действия в зависимости от выполнения условий. В языке Паскаль для записи ветвлений используются операторы if...then...else и case.


Синтаксис записи ветвлений на языке Паскаль

Основные конструкции для записи ветвлений на языке Паскаль:

  • Условный оператор if...then...else: используется для выполнения одного действия, если условие истинно, и другого действия, если условие ложно.
    • if условие then
    оператор1
else
    оператор2;
  • Оператор множественного выбора case: используется для выполнения одного из нескольких действий в зависимости от значения выражения.
    • case выражение of
    значение1: оператор1;
    значение2: оператор2;
    ...
else
    операторN;
end;

Примеры программ с ветвлениями

Пример 1: Программа для определения четности числа

program EvenOdd;
var
    num: integer;
begin
    writeln('Введите число:');
    readln(num);
    if num mod 2 = 0 then
        writeln('Число четное')
    else
        writeln('Число нечетное');
end.

Пример 2: Программа для нахождения максимального из двух чисел

program MaxOfTwo;
var
    a, b: integer;
begin
    writeln('Введите два числа:');
    readln(a, b);
    if a > b then
        writeln('Максимальное число: ', a)
    else
        writeln('Максимальное число: ', b);
end.

Пример 3: Программа для определения дня недели

program DayOfWeek;
var
    day: integer;
begin
    writeln('Введите номер дня недели (1-7):');
    readln(day);
    case day of
        1: writeln('Понедельник');
        2: writeln('Вторник');
        3: writeln('Среда');
        4: writeln('Четверг');
        5: writeln('Пятница');
        6: writeln('Суббота');
        7: writeln('Воскресенье');
    else
        writeln('Некорректный ввод');
    end;
end.

Заключение

Ветвления являются важной частью программирования, позволяя создавать гибкие и адаптивные программы. Понимание и умение использовать конструкции ветвлений на языке Паскаль позволяет решать широкий спектр задач, требующих принятия решений в зависимости от различных условий.
