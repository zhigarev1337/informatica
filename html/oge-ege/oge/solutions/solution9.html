Решение задания 9

Вопрос

На рисунке — схема дорог, связывающих города А, Б, В, Г, Д, Е, Ж, И, К. По каждой дороге можно двигаться только в одном направлении, указанном стрелкой.
Сколько существует различных путей из города А в город К, не проходящих через пункт В?

Решение

Для решения этой задачи нужно определить количество всех возможных путей из города А в город К, которые не проходят через город В.

Рассмотрим все возможные маршруты из А в К:

  • Маршрут 1: А → Б → Г → Д → К
  • Маршрут 2: А → Б → Е → К
  • Маршрут 3: А → Г → Д → К
  • Маршрут 4: А → Г → Е → К
  • Маршрут 5: А → Е → К

Все эти маршруты не проходят через город В, и их количество равно 5.

Ответ: 5
