Решение задания 3

Вопрос

Напишите наибольшее целое число x, для которого истинно высказывание:

НЕ (X <= 11) И НЕ (X >= 17).

Решение

Для решения задачи нужно понять, при каких значениях X высказывание будет истинно. Разберём каждую часть высказывания:

  • НЕ (X <= 11) означает, что X > 11
  • НЕ (X >= 17) означает, что X < 17

Таким образом, объединяя эти два условия, получаем:

  • 11 < X < 17

Наибольшее целое число, которое удовлетворяет этому условию, это 16.

Ответ: 16
