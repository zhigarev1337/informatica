Решение задания 5

Вопрос

У исполнителя Делитель две команды, которым присвоены номера:
1. раздели на 2
2. вычти 1
Первая из них уменьшает число на экране в 2 раза, вторая уменьшает его на 1. Исполнитель работает только с натуральными числами. Составьте алгоритм получения из числа 65 числа 4, содержащий не более 5 команд. В ответе запишите только номера команд.
(Например, 12112 – это алгоритм:
раздели на 2
вычти 1
раздели на 2
раздели на 2
вычти 1,
который преобразует число 42 в число 4.)
Если таких алгоритмов более одного, то запишите любой из них.

Решение

Для преобразования числа 65 в число 4 с помощью команд "раздели на 2" и "вычти 1" можно использовать следующий алгоритм:

  1. Вычитаем 1: 65 - 1 = 64
  2. Делим на 2: 64 / 2 = 32
  3. Делим на 2: 32 / 2 = 16
  4. Делим на 2: 16 / 2 = 8
  5. Делим на 2: 8 / 2 = 4

Этот алгоритм можно записать в виде последовательности команд: 21111

Ответ: 21111
