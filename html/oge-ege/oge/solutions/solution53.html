Решение задания 5

Вопрос


У исполнителя Квадратор две команды, которым присвоены номера:

1. возведи в квадрат
2. зачеркни справа

Первая из них возводит число на экране во вторую степень, вторая  — удаляет крайнюю правую цифру числа. Составьте алгоритм получения из числа 3 числа 6, содержащий не более 5 команд. В ответе запишите только номера команд.

Решение

Для получения из числа 3 числа 6 при использовании не более 5 команд необходимо выполнить команды:

  1. 1: 3^2 = 9
  2. 1: 9^2 = 81
  3. 2: удалить крайнюю правую цифру, 8
  4. 1: 8^2 = 64
  5. 2: удалить крайнюю правую цифру, 6

Таким образом, последовательность команд: 11212.

Ответ: 11212
