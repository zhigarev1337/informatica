Решение задания 5

Вопрос


У исполнителя Вычислитель две команды, которым присвоены номера:
1. вычти 1
2. умножь на 3
Первая из них уменьшает число на экране на 1, вторая утраивает его.
Составьте алгоритм получения из числа 3 числа 53, содержащий не более 5 команд. В ответе запишите только номера команд.
(Например, 12121 — это алгоритм
вычти 1
умножь на 3
вычти 1
умножь на 3
вычти 1,
который преобразует число 4 в число 23.)
Если таких алгоритмов более одного, то запишите любой из них.

Решение

Для получения из числа 3 числа 53 при использовании не более 5 команд необходимо выполнить команды:

  1. 2: 3*3 = 9
  2. 2: 9*3 = 27
  3. 2: 27*3 = 81
  4. 1: 81-1 = 80
  5. 1: 80-1 = 79
  6. 1: 79-1 = 78
  7. 1: 78-1 = 77
  8. 1: 77-1 = 76
  9. 1: 76-1 = 75
  10. 1: 75-1 = 74
  11. 1: 74-1 = 73
  12. 1: 73-1 = 72
  13. 1: 72-1 = 71
  14. 1: 71-1 = 70
  15. 1: 70-1 = 69
  16. 1: 69-1 = 68
  17. 1: 68-1 = 67
  18. 1: 67-1 = 66
  19. 1: 66-1 = 65
  20. 1: 65-1 = 64
  21. 1: 64-1 = 63
  22. 1: 63-1 = 62
  23. 1: 62-1 = 61
  24. 1: 61-1 = 60
  25. 1: 60-1 = 59
  26. 1: 59-1 = 58
  27. 1: 58-1 = 57
  28. 1: 57-1 = 56
  29. 1: 56-1 = 55
  30. 1: 55-1 = 54
  31. 1: 54-1 = 53

Таким образом, последовательность команд: 12221.

Ответ: 12221
